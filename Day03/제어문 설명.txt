제어문
 - 코드의 흐름을 제어하는 구문
 - 코드의 일정부분을 반복하거나 선택하여 실행

 - 종류
  - 조건문 : if ~ else
   - 조건식 결과에 따라서 실행내용을 선택하여 실하는 문장
  - 반볻문 : while, for
   - 조건식의 결과에 따라서 참일도안 실행내용을 반복하여 실행하는 문장

** 영역분할
 - c나 java 같은 경우 {}를 통하여 영역을 구분
 - 파이썬은 영역을 구분할때 들여쓰기를 통해서 영역을 구분한다.(들여쓰기 : tab크기)

조건문 - if ~else
 - 조건식의 결과에 따라서 실행내용을 선택하여 실행하는 문장

 - 형태에 따른 분류
  - 단순 if
   - 조건식의 결과에 따라서 실행내용을 서탣하여 실행하는 문장
   - 형식
	if 조건식 :
		종속문장
		종속문장

  - if ~ else 
   - 조건식의 결과가 참과 거짓에 따라서 실행내용을 선택하여 실하는 경우
	if 조건식: - 참
		종속문장
	else: - 위 조건이 거짓일때
		종속문장

  - if ~ elif ~ else 
   - 조건식이 여러개인 경우
   - 순차적 실행
	if 조건식 :
		종속문장
	elif 조건식 :
		종속문장
	elif 조건식 :
		종속문장
	else :
		종속문장
