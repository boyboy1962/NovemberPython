객체(object)
 - 어떠한 대사을 표현하기 위한 정보(변수)와 그 대상을 컨트롤 하기 우한 기능(함수)응ㄹ 하나의 루틴으로 묶여있는 개별적인 개체로 표현되는 변수
 - class라는 사용자 정의 자료형으로 만들어진 변수
 - python에서의 모든 데이터 타입 object

Tuple(튜플)
 - 데이터의 집합
 - 여러개의 데이터들을 저장하고 관리하는 형태
 - 데이터 변경 불가능**
 - 같은 형태와 같은 성격의 데이터들을 보다 쉽게 관리 할 수 있도록 사용한다.

 - 선언형식
   - ()를 통하여 선언하면 된다.
   - () 생략이 가능

 - 호출
   - 인덱싱, 슬라이싱을 통하여 데이터 공간을 호출하면 된다.

Packing - 자동으로 Tuple타입으로 데이터를 저장시키는 것... ()생략
Upacking - 자동으로 Tuple타입으로 데이터를 변수 타입으로 저장시키는 것..

Tuple에서 사용되는 함수들...

count(value)
 - 투플에서 일치하느는 값의 수를 반환하는 함수
 - 없으면 0 있으면 개수를 반환

index(value)
 - 투플에서 이치하는 값의 인덱스 번호를 반환하는 함수
 - 데이터가 없으면 error
