List
 - 데이터의 집합
 - 여러게의 데이터들을 저장하고 관리하는 형태
 - 데이터의 변경이 가능하다.***

 - 선언형식
   - []를 통하여 선언하면 된다.
   - 변수명 = [ 데이터, 데이터, 데이터... ]

 - 호출방식
   - 인덱싱과 슬라이싱 모두 사용 가능
   - 변수명[인덱스]
	- index: 0번째부터 저장된 데이터 개수보다 -1

list에서 자주 사용되는 함수
append(value)
 - 리스트의 끝에 값을 추가하는 함수

#iter - 다량의 데이터를 가지고 있는 데이터의 형식.. list, tuple, dic, str
extend(iter)(
 - 리스트 끝에 list, tuble, dic, str의 값을 하나씩 추가하는 함수
	ex) (5,6) -> 5, 6 한개씩 데이터를 뽑아서 저장

insert(idx,value)
 - 특정 위치에 값을 추가하는 함수
 - 인덱스 범위가 벗어나면 제일 끝에 데이터를 추가한다.

pop(idx)
 - 기복적으로 마지막 인덱스 안에 있는 값을 반환하고 삭제하느 함수( 인덱스를 적지 않을때 )
 - 인덱스 값을 적으면 인덱스 위치의 데이터를 삭제한다.
 - 인덱스 범위를 벗어나면 error

remove(value)
 - 특정 값에 해당되는 것을 찾아 삭제하느 함수
 - 값이 없으면 error

clear()
 - 모든 값을 삭제하여 빈 리스트만 남기는 함수

count(value)
 - 리스트에서 일치하느 값의 수를 반환하는 함수

index(value)
 - 리스트에서 일치하는 값의 인덱스 번호를 반환한다.

reverse()
 - 리스트의 모든 값을 뒤집어 나열하는 함수

sort(reverse = True/False)
 - 리스값의 값을 오름차순(False) 생략가능
 - 내림차순 True
